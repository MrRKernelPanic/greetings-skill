Adds some responses to common greetings

## About
Hopefully this skill will allow mycroft to answer simple greets with some sort of response, e.g. hello, goodbye or good afternoon.

## Examples
* "Hello"
* "Goodbye"
* "Bye"
* "Good morning"
* "Good afternoon"
* "Good evening"
* "How are you"
